Example Nest to AWS


## Running

```
yarn install
yarn start:dev
```

## Using Postman
Using multipart-form format for body </br>
### Store to S3
`http://localhost:3000/uploads3` for upload to your S3 </br>

### Store in Local
`http://localhost:3000/upload` for store local folder `upload`
`http://localhost:3000/myfile.svg` to get the data
